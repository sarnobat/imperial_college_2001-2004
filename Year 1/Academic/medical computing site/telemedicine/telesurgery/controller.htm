Controller

The Controller architecture’s main components. The Supervisor processes user (i.e. surgeon) input (UI is a finite state machine); the Middleman handles most of the mathematical calculations (like an ALU); there are several MSD (Multiple Servo Drive) boards for controlling amplifiers; the FRL is a Fault Reaction Logic unit which the Health Monitoring Network makes use of when logging errors.
