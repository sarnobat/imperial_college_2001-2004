Medical Image Visualization and Analysis

With advances in technology of medical imaging, the display and analysis of medical images is becoming more and more important. Medical imaging is becoming one of the most important sources of information for medical practitioners and as a result techniques of image visualization and analysis have to be sophisticated.

Visualization techniques

See visulization in imaging for further information

Research is done in this area to develop visualization allowing optimal (usually 3D) representation of images. The use of virtual and augmented reality is also becoming increasingly important in this field.

As different imaging modalities are being developed, visualization of multi-parameter medical image data sets is becoming more important. It is usually desirable to map all these data sets into a single gray-scale image. For this purpose the Self Organizing-map (SOM) visualization technique is used. This is a neural network that maps data from n-dimensional space into a lattice of nodes where each node represents a reference vector. Maps can be of any dimensions less than n, though usually SOM applications use two-dimensional maps.

The SOM can in fact be thought of as an ordered array of nodes. A gray scale can thus be mapped along it, hence generating a single image which combines information of all component images.

Segmentation

Manual Segmentation

  • What is most usual but not always suitable
  • Problems arise when there are many organs identified in an image.
  • Amount of work is greatly increased and there is poor reproducibility of results.

    • A complete segmented 3D model of a female abdomen

  • Automatic Segmentation

    This technique is more useful the manual segmentation. In earlier years, it was done based on the assumption that radiological images were self-contained, which is not true. As a result the technique worked in only limited applications because in fact radiologists use related knowledge in different fields to analyse an image. Research is being done to find ways to code and use such knowledge for image segmentation.

  • More generic solutions

    Automatic segmentation generally works only for a narrow range of radiological data. More generic platforms are desirable and these can be expected by developing interactive 3-dimensional segmentation systems which allow cooperation between image analysis algorithms and human operators.

    MIPAV – Medical Image Processing Analysis and Visualization

    This is a program developed to process image data from most medical imaging modalities. It has the following properties:
      1. N-dimensional
      2. Platform independent
      3. It can process data from most medical image modalities
      4. It can function as an end-user application or as an Application Programming Interface (API).
      5. It is written in JAVA and is modular in design.
      6. Its data-type buffer can store and process images of up to four dimensions.
      7. One of its modules contains over 20 image processing and analysis tools.
