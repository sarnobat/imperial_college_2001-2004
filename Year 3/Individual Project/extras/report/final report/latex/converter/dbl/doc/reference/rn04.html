1. Introduction

This reference describes the templates and parameters relevant to formatting DocBook bibliography markup.

2. All Vs. Cited mode

Using this option, only the biblioentries that have been cited somewhere in the document are output. Otherwise (in All mode) all bibentries found are output (as the HTML stylesheets do).
