Name

revhistory — revhistory XSL template

Synopsis

<xsl:template match="revhistory"/>

User variables. 

  • latex.output.revhistory

This XSL template depends on the value of latex.output.revhistory. If this variable is "1", the XSL template calls map.begin, then applies templates and finally calls map.end
