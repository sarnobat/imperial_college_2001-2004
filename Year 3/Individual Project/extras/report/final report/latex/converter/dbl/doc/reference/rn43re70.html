Name

latex.graphics.formats — Control imagedata selection.

Synopsis

<xsl:param name="latex.graphics.formats"/>

This controls how DB2LaTeX behaves when a mediaobject contains multiple imagedata. When non-empty, DB2LaTeX will exclude imagedata that have a format no listed within this variable.
