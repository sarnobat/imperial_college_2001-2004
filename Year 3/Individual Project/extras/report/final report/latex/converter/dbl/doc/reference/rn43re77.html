Name

latex.documentclass.pdftex — DB2LaTeX document class options for pdfTeX output

Synopsis

<xsl:param name="latex.documentclass.pdftex"/>

The pdfTeX options are empty by default. These are the last options to be passed to \documentclass and override the common/article/book options.
