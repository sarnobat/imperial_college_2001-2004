Name

attribution — This template produces no output.

Synopsis

<xsl:template match="attribution"/><xsl:template match="attribution" mode="block.attribution"/>

The attribution element only occurs within blockquote and epigraph. However, the templates for those elements use a “mode” mechanism. Therefore, this template is intentionally suppressed and a replacement exists. See attribution (in block.attribution mode) instead.
