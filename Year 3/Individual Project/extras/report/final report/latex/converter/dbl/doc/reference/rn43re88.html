Name

latex.use.fancyvrb — Toggle the use of the fancyvrb™ LaTeX package

Synopsis

<xsl:param name="latex.use.fancyvrb">1</xsl:param>

Provides support for tabbed whitespace in verbatim environments. See also latex.fancyvrb.tabsize.
