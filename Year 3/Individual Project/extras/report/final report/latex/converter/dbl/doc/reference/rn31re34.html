Name

chapter — XSL template for Chapters.

Synopsis

<xsl:template match="chapter"/>

This is the main entry point for a chapter subtree. This template processes any chapter. Outputs \chapter{title}, calls templates and apply-templates. Since chapters only apply in books, some assumptions could be done in order to optimize the stylesheet behaviour.

Remarks and Bugs. 

  • EMPTY templates: chapter/title, chapter/titleabbrev, chapter/subtitle, chapter/docinfo|chapterinfo.
