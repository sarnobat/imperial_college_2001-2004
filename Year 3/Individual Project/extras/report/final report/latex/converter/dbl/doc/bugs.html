Known Bugs

  • Chunking does not work.

  • Table Support is partially implemented.

  • Lists column management , half - fixed.

  • When the user specifies a preamble or overrides some templates, there may exist undefined commands (e.g. \hypertarget without the hyperref package). Check.

  • Extra spaces generated at the end of segtitle.

  • \usepackage{fancybox} prevents toc and lof, even added by hand, to be generate: we get empty shells (try it directly in the TeX).

  • biblioset is not supported (Workaround, need to work on this).

  • biblioentry have funny final punctuation (dangling commas and the like).

  • character not supported in the text Check out all iso-latin-1 while you're at it.

  • Feature requests: . I would like the thing to handle languages in documents (article lang="fr". Right now I add by hand the needed.

  • \usepackage for input encoding and active characters.

  • Aesthetics remarks: sometimes you use {} and sometimes not in the preamble (\usepackage).

  • itemizedlists inside tables will cause LaTeX errors unless column widths are specified (but tabularx is fine).

Low Priority Bugs

  • Localization: LaTeX localization (Chapter, etc) and book lang="fr" may not be coherent.
