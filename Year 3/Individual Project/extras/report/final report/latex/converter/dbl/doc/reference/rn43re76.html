Name

latex.documentclass.book — DB2LaTeX document class options for books

Synopsis

<xsl:param name="latex.documentclass.book">a4paper,10pt,twoside,openright</xsl:param>

The book options are set to a4paper,10pt,twoside,openright by default. These are the intermediate options to be passed to \documentclass, between the common options and the pdftex/dvips options.
