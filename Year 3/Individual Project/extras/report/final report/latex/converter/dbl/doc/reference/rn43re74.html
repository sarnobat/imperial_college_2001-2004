Name

latex.documentclass.common — DB2LaTeX document class options

Synopsis

<xsl:param name="latex.documentclass.common">french,english</xsl:param>

These are the first options to be passed to \documentclass The common options are set to french,english by default. They will be augmented or superseded by article/book options (see $latex.documentclass.article and $latex.documentclass.book) and pdftex/dvips options (see $latex.documentclass.pdftex and $latex.documentclass.dvips).
