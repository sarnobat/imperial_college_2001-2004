Name

use.role.for.mediaobject — Control mediaobject selection methods

Synopsis

<xsl:param name="use.role.for.mediaobject">1</xsl:param>

This controls how DB2LaTeX behaves when a figure contains multiple mediaobjects. When enabled, DB2LaTeX will choose the mediaobject with the "LaTeX" or "TeX" role, if present. Otherwise, the first mediaobject is chosen.
