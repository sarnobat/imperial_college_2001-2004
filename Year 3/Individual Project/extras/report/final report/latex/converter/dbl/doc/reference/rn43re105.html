Name

latex.figure.title.style — Control the style of figure titles

Synopsis


	    

Figure titles are typeset in the default typeface (usually 'roman') but if
		this option is set to a LaTeX command, such as \itshape{}, then
		that command will precede the title and it will be typeset accordingly.
