Name

abstract — Abstract XSL template.

Synopsis

<xsl:template match="abstract"/>

Calls template ???(map.begin), to output the opening command (by default):

		% --------------------------------------------
		% Abstract 
		% --------------------------------------------
		\begin{abstract}

Calls "apply-templates" for all the children, and call template ???(map.end) to output the closing command.

Remarks and Bugs. 

  • The title of the abstract is lost.

  • The template for abstract/title is defined EMPTY.

Default Behaviour. 

		    Fill
