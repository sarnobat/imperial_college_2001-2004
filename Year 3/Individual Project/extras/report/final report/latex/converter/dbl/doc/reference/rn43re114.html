Name

latex.article.begindocument — The begin document

Synopsis

The value of this variable is output from the article template
	    article after the author command. The default value
	    (shown below) is just the begin document command. Users of the XSL LaTeX
	    stylesheet may override this parameter in order to output what they want.
	
Returns\begin{document}
