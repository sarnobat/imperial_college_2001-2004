Name

admon.graphic — Choose an admonition graphic

Synopsis

<xsl:template name="admon.graphic">
<xsl:param name="node" select="."/>
  ...
</xsl:template>

For each admonition element (note, warning, caution, top, important), this template chooses the graphics filename. If the admonition element is not known, the note graphic is used.
