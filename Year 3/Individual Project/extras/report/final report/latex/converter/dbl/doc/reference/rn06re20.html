Name

article — Article XSL Template

Synopsis

<xsl:template match="article">
<xsl:variable name="article.title">
	    <xsl:choose>
			<xsl:when test="./title"> 
				<xsl:value-of select="./title"/>
			</xsl:when>
			<xsl:when test="./articleinfo/title">
				<xsl:value-of select="./articleinfo/title"/>
			</xsl:when>
			<xsl:otherwise>
				<xsl:value-of select="./artheader/title"/>
			</xsl:otherwise>
	    </xsl:choose>
	</xsl:variable>
  ...
</xsl:template>

Most DocBook documents are either articles or books, so the article XSL template article is one classical entry point when processign docbook documents.

Tasks. 

  • Calls generate.latex.article.preamble.

  • Outputs \title, \author, getting the information from its children.

  • Calls latex.article.begindocument.

  • Calls latex.article.maketitle.

  • Applies templates.

  • Calls latex.article.end template.

Remarks and Bugs. 

  • EMPTY templates: article/title and article/subtitle

Parameters
colwidth

The CALS column width specification.

Returns

Outputs the LaTeX Code corresponding to an article.
