Name

revhistory/revision — revhistory/revision XSL template

Synopsis

<xsl:template match="revhistory/revision">
<xsl:variable name="revnumber" select=".//revnumber"/>
<xsl:variable name="revdate" select=".//date"/>
<xsl:variable name="revauthor" select=".//authorinitials"/>
<xsl:variable name="revremark" select=".//revremark|../revdescription"/>
  ...
</xsl:template>

Each revhistory/revision corresponds to a LaTeX table row (see revhistory) containing the revision number, the date, author initials and the description/ remarks of the revision.
