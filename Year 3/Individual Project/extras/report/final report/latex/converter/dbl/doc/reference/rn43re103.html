Name

latex.pagestyle — Override DB2LaTeX's choice of LaTeX page numbering style

Synopsis


	    

By default, DB2LaTeX will choose the 'plain' or 'fancy' page styles,
		depending on latex.use.fancyhdr. If non-empty, this
		variable overrides the automatic selection. An example would be the literal
		string 'empty', to eliminate headers and page numbers.
