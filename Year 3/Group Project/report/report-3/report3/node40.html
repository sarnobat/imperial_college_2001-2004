Console Output

The Console I/O box is used for 2 things: - Error messages from the parser - if an error was encountered while parsing the assmebly file, the type of error and related line number are displayed. Correct the error then try loading the file again.

- Output of the program - during the execution of the program, any output is displayed in the Console I/O box.
